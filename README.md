# streamani.net-downloader
is a batch downloader / link generator for  genoanime.com

### Setup
1. `git clone "https://github.com/shahrier-nafis/streamani.net-downloader.git"`
1. `cd streamani.net-downloader`
1. `pip install -r requirements.txt`

### features
- get video links in .mp4
- get video links in .m3u8
- get all links in .txt file

###usage
1. `python cli.py`
1. enter first episode link [ex:** https://streamani.net/videos/boku-no-hero-academia-5th-season-dub-episode-13**]
1. Enter Episode Number to start with :`1`
1. Enter Episode Number to end with :`14`
1. quality:`Enter`
1. filetype:`Enter`
